### 컴포넌트 스캔과 자동 의존 관계 설정

- 스프링 빈을 설정하는 두 가지 방법
    - 컴포넌트 스캔과 자동 의존 관계 설정
        - @service, @Repository 등등
    - 자바 코드로 직접 스프링 빈 등록하기

- @Component : 애노테이션이 있으면 스프링 빈으로 자동 등록
- @Controller : 컨트롤러가 스프링 빈으로 자동 등록된 이유는 컴포넌트 스캔
    
    
- @Component를 포함하는 다음 애노테이션도 스프링 빈으로 자동 등록
    - @Controller
    - @Service
    - @Repository
